# Feedback

We appreciate all the feedback we receive. If you experience any issues when using the Superface CLI or OneSDK, you can let us know about these by raising issues on their respective GitHub repositories.

[Raise an issue with OneSDK](https://github.com/superfaceai/one-sdk/issues) | [Raise an issue with Superface CLI](https://github.com/superfaceai/cli/issues)

## GitHub Discussions

You are all welcome to join us on [GitHub Discussions](https://github.com/orgs/superfaceai/discussions) where you can ask questions about anything Superface, API or AI related. It's a great space to share your ideas about the features and improvements you'd like to see made with the CLI and OneSDK in the future.
