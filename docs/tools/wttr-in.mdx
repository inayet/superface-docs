# Wttr.in

The Wttr.in tool for Superface allows you to quickly add the following capabilities:

- Retrieve current weather information for a specified location.

## Requirements

There are no requirements for using Wttr.in. It is freely available.

## Authentication

Authentication is not required when using Wttr.in.

:::note Import your schema

Remember to re-import the schema in your GPT to update it with the new capabilities added by this tool.

:::
