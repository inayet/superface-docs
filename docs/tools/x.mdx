# X (Twitter)

The X Tool for Superface allows you to quickly add the following capabilities:

- Create and publish a new tweet using the Twitter API under the authorized account.

## Authentication

In order to use the X tool you must connect the Twitter/X account that you want to publish with.

![Sign in with Twitter/X to connect your account](/img/tools/tools-x-signin.png)

You will be redirected to the X login screen. Enter your credentials and log in. You will then be asked to authorize Superface with specific permissions to post from your account.

![Authorization screen on X](/img/tools/tools-x-authorize.png)

Click on _Authorize app_. You will then be redirected back to Superface and your account will now be connected and ready to use.

:::note Import your schema

Remember to re-import the schema in your GPT to update it with the new capabilities added by this tool.

:::
