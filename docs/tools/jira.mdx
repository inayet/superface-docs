# Jira

The Jira tool for Superface allows you to quickly add the following capabilities:

- Current User Information
- Dashboard Management
- Group Picker Suggestions
- Issue Search
- Project Visibility

## Requirements

In order to use the [Jira](https://atlassian.com) tool you will first need to register an Atlassian account.

## Authentication

To use the Jira tool with Superface you will need:

- The `DOMAIN` for your account. For example, the domain of `https://superface.atlassian.net` is `superface`.
- The `username` and `password` for your Atlassian account.

![Enter your domain, username and password.](/img/tools/tools-jira-credentials.png)

Click on Save Changes to update the tool with the new credentials.

:::note Import your schema

Remember to re-import the schema in your GPT to update it with the new capabilities added by this tool.

:::

## Example prompts

To test that everything is working as it should be, from your GPT or the Superface Agent you can try asking:

- List the Jira issues that are assigned to me
- List the Jira tasks that have not been updated for over a week
