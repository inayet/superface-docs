# World Time API

The World Time API tool for Superface allows you to quickly add the following capabilities:

- Request the current date and time based on the IP address of the request.

## Requirements

There are no requirements for using the World Time API. It is freely available.

## Authentication

Authentication is not required when using the World Time API.

:::note Import your schema

Remember to re-import the schema in your GPT to update it with the new capabilities added by this tool.

:::
