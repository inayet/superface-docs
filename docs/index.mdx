# Welcome to Superface

Superface is an AI powered agent that helps small to medium-sized companies make informed decisions by consolidating real-time data from multiple sources to answer queries posed using natural language.

Superface connects to your systems and SaaS applications and works in real-world scenarios. This is achieved by the use of custom "tools" that can be created to connect to any service that has an API.

In addition, Superface is the first to offer business task automation solely using AI so that you can automate your daily tasks without any technical knowledge.
